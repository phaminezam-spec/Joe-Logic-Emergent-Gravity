* The Stressed Regime ($z > 5.85$):
* Network: Becomes an over-constrained network where redundant edges ($R$) accumulate.
   * Spacetime: This is where Emergent Gravity appears. The "metric" locks in, and stress-energy is encoded by the redundant connections ($R$), creating "curvature hotspots."

2. The Critical Transition: The Isostatic Point
The boundary between these two states is the Isostatic Point ($z_c \approx 5.85$). This is the critical threshold where the network's degrees of freedom ($F$) drop to zero ($F \to 0$), and the system transitions from a structurally unstable "fluid" state to a stable, Rigid Metric Spacetime.
